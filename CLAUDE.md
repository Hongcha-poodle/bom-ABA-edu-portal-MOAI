# bom-ABA-edu-portal 지침

이 문서는 프로젝트 개발을 위한 기본 지침을 담고 있습니다.

## 1. 핵심 기술 스택
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- UI Components: Lucide React

## 2. 개발 가이드라인
- 모든 설명과 코멘트는 한국어로 작성합니다.
- 기술 용어 및 프레임워크 이름은 영문을 유지합니다.
- `aba-edu-portal` 디렉토리가 메인 소스코드 위치입니다.
